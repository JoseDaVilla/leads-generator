Google Maps Business Scraper

This implementation demonstrates how to scrape business data from Google Maps using Puppeteer.

Note: This scraper is for educational purposes. Always respect websites' terms of service and robots.txt files. Web scraping may violate terms of service of some websites and could potentially lead to IP blocking.

Implementation Example

The scraper performs the following steps:

  1. Initializes a Puppeteer browser with stealth plugin to avoid detection
  2. Navigates to Google Maps
  3. Searches for a specific category in a location
  4. Scrolls through results to load more businesses
  5. Extracts business information (name, address, phone, rating, etc.)
  6. Saves data to the database

Sample Code

// Example of how to use the scraper
const googleMapsScraper = require('./services/googleMapsScraper');

async function scrapeCategory(category, location) {
  try {
    await googleMapsScraper.initialize();
    
    const searchQuery = `${category} in ${location}`;
    const options = { maxResults: 20 };
    
    const businesses = await googleMapsScraper.scrapeBusinesses(searchQuery, options);
    
    console.log(`Found ${businesses.length} businesses for ${category} in ${location}`);
    console.log(businesses);
    
  } catch (error) {
    console.error(`Error: ${error.message}`);
  } finally {
    await googleMapsScraper.close();
  }
}

// Example usage
scrapeCategory('Coffee Shops', 'New York, NY');

Resultados

Algunos resultados de la búsqueda de hoteles y alquileres de vacaciones podrían personalizarse según tu actividad de navegación y tus búsquedas recientes en Google, así como las confirmaciones de viaje que se envían a tu cuenta de Gmail.

Los precios provienen de socios de Google.

Todas las opiniones sobre alquileres de vacaciones y algunas opiniones sobre hoteles provienen de sitios de viajes de terceros. Las opiniones sobre alquileres de vacaciones se muestran como calificaciones combinadas de las opiniones de terceros vinculadas. Google no verifica las opiniones, pero sí revisa que el contenido no sea falso, y lo quita cuando lo identifica como tal en las opiniones en Google. Para consultar las políticas de opiniones de sitios de viajes de terceros, visita sus sitios.
